== Description ==

Beatrix Lite is a beautiful WordPress blog theme for personal blogs. Beatrix Lite has a modern, clean and elegant look and lots of customization for bloggers.

Built on the latest technology of WordPress, Beatrix Lite is lightweight, fast and optimized for all mobile phones.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload beatrix-lite.zip file.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel.
